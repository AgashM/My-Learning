**ENCAPSULATION IN JAVA**

Encapsulation is the bundling of data and methods that work on the data into a single unit, like a class.

The Encapsulation, mechanism protects the internal state of programmer object and allows for better control

**How Encapsulation achieved in Java**

1. Private data members
2. public getter and setter methods

Key Rules:
* Declare data as private
* Use Getters and Setter
* Apply proper access modifiers

**Advantages of Encapsulation**

1. Data hiding
2. Improved Maintainability
3. Enhanced Security
4. Code Reusability
